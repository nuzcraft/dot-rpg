## Controls

- Arrow Keys to move around
- Mouse or Arrow Keys + Enter for menus
- ESC to Pause
- S for the Hero Stats Page
- _Touch Controls on Mobile supported (not fully tested)_
- _Joypad Controls supported (not fully tested)_
